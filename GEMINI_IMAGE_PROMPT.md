# Промпт для генерації зображень для EventBooking сайту

## Загальний стиль та дизайн

Створюй зображення для веб-сайту бронювання квитків на події (EventBooking). Сайт має сучасний, мінімалістичний дизайн з градієнтами та м'якими формами.

## Кольорова палітра

**Основні кольори:**
- Primary (основний): Синьо-блакитний (#0ea5e9) - яскраве небо, чиста вода
- Secondary (додатковий): Бірюзовий/циан (#06b6d4) - тропічні води, небо на світанку
- Accent (акцент): Рожево-червоний (#f43f5e) - енергійний, живий колір

**Фон:**
- Світла тема: Світло-блакитний градієнт (#f0f9ff до #e0f2fe)
- Темна тема: Темно-синій градієнт (#0c1222 до #0f172a)

**Текстові кольори:**
- Основний текст: Темно-сірий (#1e293b) / Світло-сірий (#f1f5f9) для темної теми
- Вторинний текст: Сірий (#64748b) / Світло-сірий (#cbd5e1)

## Стиль оформлення

- **Сучасний мінімалізм**: Чистий дизайн, багато білого простору
- **Градієнти**: Плавні переходи між синьо-блакитними відтінками
- **М'які форми**: Закруглені кути (rounded corners), округлі елементи
- **Глибина**: Легкі тіні, backdrop blur ефекти, шаруватість
- **Інтерактивність**: Hover ефекти, анімації, плавні переходи

## Технічні вимоги до зображень

- **Розмір**: Оптимально 1920x1080px або більше для hero зображень
- **Формат**: JPG або WebP
- **Якість**: Високоякісне зображення, чіткі деталі
- **Співвідношення**: Для hero - landscape (16:9), для карток - квадрат або 4:3
- **Оптимізація**: Для веб, але з високою роздільною здатністю

## Загальні принципи генерації

1. **Атмосфера**: Позитивна, енергійна, життєрадісна
2. **Освітлення**: Природне або яскраве сценічне освітлення
3. **Композиція**: Центральна фокусна точка, правило третин
4. **Стиль**: Сучасний, професійний, чистий
5. **Кольори**: Переважно синьо-блакитна палітра з рожевими акцентами
6. **Люди**: Щасливі, активні, взаємодіючі з подіями
7. **Деталі**: Достатньо деталей, але не перевантажені

## Приклади промптів для різних секцій

### Hero зображення альтернатива:
```
Широкий план великого концертного залу або стадіону з тисячами людей. Яскраве сценічне освітлення створює градієнт від блакитного до бірюзового. Енергійна атмосфера музичного фестивалю. Професійна фотографія, високий контраст, яскраві кольори. Стиль: епічний, масштабний, вражаючий.
```

### Картки функцій (Features):
```
Мінімалістичні іконографічні зображення: білет на концерт на блакитному градієнтному фоні, телефон з додатком бронювання, замок символізуючий безпеку, зірки для рейтингів. Плоский дизайн, м'які тіні, закруглені кути. Кольори: переважно блакитний та бірюзовий з акцентами.
```

### Фонові елементи:
```
Абстрактні форми та градієнти в синьо-блакитних тонах. М'які, розпливчасті форми. Прозорість та легкість. Використовується для декоративних елементів та фонів. Стиль: сучасний, абстрактний, елегантний.
```

## Додаткові рекомендації

- Уникай надто темних або похмурих кольорів
- Перевагу віддавай природному освітленню
- Включи елементи української культури (якщо релевантно)
- Зберігай консистентність кольорової палітри
- Зображення мають бути універсальними та зрозумілими
